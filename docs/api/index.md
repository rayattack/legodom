## Browser Support

Lego requires:
- Web Components (Custom Elements v1)
- Shadow DOM v1
- ES6 Proxy
- Template Literals
- MutationObserver

Supported browsers:
- Chrome 63+
- Firefox 63+
- Safari 11.1+
- Edge 79+
